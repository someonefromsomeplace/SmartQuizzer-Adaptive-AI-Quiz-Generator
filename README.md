# SmartQuizzer – Adaptive AI-Based Quiz Generator

SmartQuizzer is an AI-powered adaptive quiz generation system designed for e-learning platforms. 
The application analyzes user-provided study materials and automatically generates quiz questions 
using Large Language Models (LLMs). The system dynamically adapts question difficulty based on 
user performance and provides detailed learning analytics.

## Key Features
- Upload PDF or text-based study materials
- AI-driven concept extraction and question generation
- Adaptive difficulty adjustment based on user responses
- Topic-wise performance analytics and visualizations
- Streamlit-based interactive web application

## Tech Stack
- Python
- Large Language Models (OpenAI / HuggingFace)
- Natural Language Processing (NLP)
- Streamlit
- Plotly & Matplotlib
- JSON / MongoDB (optional)

## Project Workflow
1. Document upload and text extraction
2. Concept extraction using LLMs
3. AI-based question generation
4. Difficulty classification
5. Adaptive quiz delivery
6. Performance analytics and recommendations
